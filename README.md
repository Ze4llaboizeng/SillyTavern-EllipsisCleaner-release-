### 📄 ไฟล์ `README.md` (สำหรับนำไปใช้งาน)

# SillyTavern-EllipsisCleaner

Extension สำหรับจัดการเครื่องหมายจุดไข่ปลา (Ellipsis) หรือ "..." ที่ AI มักสร้างออกมามากเกินความจำเป็น พร้อมเครื่องมือทำความสะอาดข้อความส่วนเกินอื่นๆ เพื่อให้อ่านง่ายและเป็นธรรมชาติมากขึ้น

## ✨ ฟีเจอร์หลัก (Features)

* **ลบจุดไข่ปลาอย่างชาญฉลาด:** ลบ `...`, `…` อัตโนมัติ (มีตัวเลือกสำหรับลบ `..` หรือลบจุด `.` ทั้งหมดได้)


* **Auto Remove & Clean Depth:** ฟีเจอร์ทำความสะอาดอัตโนมัติ โดยสามารถกำหนดความลึก (`Auto Clean Depth`) เช่น ตรวจจับเฉพาะ 1 ข้อความล่าสุด เพื่อไม่ให้รบกวนข้อความเก่า


* **Protect Code & HTML:** ป้องกันไม่ให้ Extension ลบจุดภายในแท็ก `<think>`, บล็อกโค้ด (```), หรือแท็ก HTML (`<script>`, `<style>`)


* **Remove English in ( ):** ตัวเลือกพิเศษสำหรับลบวงเล็บภาษาอังกฤษที่โผล่มาหลังคำไทย (เช่น `แชท(chat)` จะเหลือเพียง `แชท`)


* **ระบบ Undo:** กดย้อนกลับ (Undo) เพื่อยกเลิกการแก้ไขข้อความครั้งล่าสุดได้


* **Per-message & Quick UI:** มีปุ่มยางลบในแต่ละข้อความ และปุ่มเมนูด่วนข้างช่องแชท (สามารถกดคลิกหรือกดค้างเพื่อดูตัวเลือก)

## ⚙️ วิธีติดตั้ง (Installation)

1. เปิด SillyTavern ไปที่เมนู **Extensions** (รูปปลั๊กไฟ)


2. เลือก **Install from URL**

3. วางลิงก์ GitHub Repo ของส่วนขยายนี้ แล้วกดติดตั้ง


4. เปิดใช้งาน **Ellipsis Cleaner** จากหน้า Extensions ได้เลย



---

## 📖 คู่มือการใช้งาน (How to Use)

Extension นี้ออกแบบมาให้ใช้งานง่าย และลดการสลับหน้าจอไปมา:

1. **การใช้งานปุ่มด่วน (Quick Access Button):**
ที่ด้านข้างของช่องพิมพ์ส่งข้อความ จะมีปุ่มไอคอนยางลบแสดงอยู่
* **คลิก 1 ครั้ง:** สั่งทำงานลบจุดไข่ปลาตามการตั้งค่าปัจจุบันทันที
* **กดค้าง (Long Press) / คลิกขวา:** เปิดเมนู Popup ด่วน เพื่อใช้คำสั่งอื่นๆ เช่น Undo, เช็คจุดไข่ปลา, หรือสลับเปิด-ปิดโหมด Auto


2. **การลบเฉพาะข้อความ (Per-Message Cleaning):**
หากต้องการแก้ไขเฉพาะบางข้อความ ให้คลิกที่ไอคอน "ยางลบ" ที่อยู่ในแถบเมนูจัดการของข้อความนั้นๆ (บริเวณเดียวกับปุ่ม Edit/Delete) เพื่อทำความสะอาดแค่กล่องข้อความนั้นกล่องเดียว
3. **การตั้งค่าความลึก (Auto Clean Depth):**
ไปที่เมนูตั้งค่าหลัก แนะนำให้กำหนดค่าไว้ที่ `1` ระบบจะลบจุดไข่ปลาเฉพาะกล่องข้อความที่ AI เพิ่งตอบกลับมาล่าสุดเท่านั้น หากคุณตั้งค่าเป็น `0` ระบบจะกวาดล้างจุดไข่ปลาในทุกข้อความบนหน้าแชท
4. **หากระบบลบผิดพลาด (Undo):**
ไม่ต้องตกใจ หากการลบอัตโนมัติทำให้เนื้อหาบางส่วนหายไป สามารถเปิดเมนูด่วนแล้วกดคำสั่ง **Undo** ข้อความจะเด้งกลับมาเป็นเวอร์ชันดั้งเดิมก่อนถูกเคลียร์ทันที

## 🎨 การปรับแต่งหน้าตา (Styling)

Extension นี้ได้รับการออกแบบ UI ใหม่ให้ดึงค่าสีของ SmartTheme ในระบบ SillyTavern มาใช้โดยอัตโนมัติ ทำให้กลมกลืนไม่ขัดตา ทั้งบน Desktop และ Mobile
หากต้องการปรับแต่งขนาดของปุ่มหลักด้วยตนเอง สามารถเข้าไปแก้ตัวแปรได้ในไฟล์ `style.css`:

* `--rm-ell-btn-size` (สำหรับขนาดบนหน้าจอคอมพิวเตอร์)


* `--rm-ell-btn-size-mobile` (สำหรับขนาดบนหน้าจอมือถือ)
